django-squash
===========

.. image:: https://travis-ci.com/kingbuzzman/django-squash.svg?branch=develop
    :target: https://travis-ci.com/kingbuzzman/django-squash

django-squash is a migration squashing replacement based on a internal need that occured in my dayjob at `EAB Inc. <https://www.eab.com/>`_.


```
pip install -e '.[test]'
python setup.py test
```

```
coverage run setup.py test
coverage report -m
```
